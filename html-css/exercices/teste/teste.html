Testes de formatações

Itálico/ênfase

Esse texto pode ser em itálico (mudança de forma). <i>

Esse texto pode ser em ênfase (mudança de significado). <em>

Negrito/destaque

Podemos negritar palavras usamos o <b> (mudanças de forma),

Podemos destacar palavras, usamos o <strong>

Texto pequeno/grande

Para deixar o texto grande, usamos <big> (não semântico e em desuso).

Para deixar um texto pequeno, usamos <small>.

Marcação de texto

Não é um comando identificado no Visual Studio, porém ele existe, e é semântico. <mark>

Podemos mudar a cor do mark usando <style> dentro do comando.

Texto inserido

Esse é o texto inserido (semântico). <ins>

Esse é o texto com sublinhado (não semântico). <u>

Texto deletado

Aquele texto com um risco no meio você faz com <del>

Texto sobrescrito

x8 + 123 para deixar o número elevado, usamos o comando <sup>

Texto subscrito

H20 para deixar um número embaixo e pequeno, usamos o comando <sub>
